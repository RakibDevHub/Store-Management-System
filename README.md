# Store Management System

A simple PHP-based Store Management System for managing inventory, sales, and generating reports.

## Features
- User Authentication (Admin/Staff)
- Category Management
- Product Management
- Sales Recording
- Sales Reports

## Tech Stack
- PHP 7.4+
- MySQL
- HTML/CSS (No frameworks)

## Installation
1. Import database.sql into MySQL
2. Configure database in config/database.php
3. Default Admin: admin@store.com / admin123
